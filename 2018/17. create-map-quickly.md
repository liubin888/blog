想要快速创建 `Map` ，不用频繁 `new` ，最快的方法就是用 `Guava` ，使用 `ImmutableMap.of("a", 1, "b", 2, "c", 3);`

### Guava

```java
Map<String, Integer> left = ImmutableMap.of("a", 1, "b", 2, "c", 3);
```

### java 9

```java
Map<Integer, String> map = Map.of(1, "A", 2, "B", 3, "C");
```

超过 10 组会不支持，那么就要这样：

```java
Map.ofEntries(
    Map.entry( 1, false ),
    Map.entry( 2, true ),
    Map.entry( 3, false ),
    Map.entry( 4, true ),
    Map.entry( 5, false ),
    Map.entry( 6, true ),
    Map.entry( 7, false ),
    Map.entry( 8, true ),
    Map.entry( 9, false ),
    Map.entry( 10, true ),
    Map.entry( 11, false )
);
```
